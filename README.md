This repository is purely for making documentation for Qt User Intarface from Black Mesa game.

The main doc: https://developer.valvesoftware.com/wiki/Qt_(Black_Mesa)

Notes:
* Folder "bin" contains DLLs purely related to the UI.
* This repository contains multiple UI versions:
** bms - the original UI from "Black Mesa" game.
** bshift - the UI from "Black Mesa: Blue Shift" modification.
** hcad - the UI from "Black Mesa: Hazard Course After Disaster - Restored" modification.
